시멘틱 태그란?

시멘틱 태그는 HTML5에서 도입된 태그로, 웹 페이지의 구조를 명확하게 표현하는 데 사용됩니다. 시멘틱 태그는 태그의 의미를 명확히 하여 웹 접근성과 SEO에 도움이 됩니다.

주요 시멘틱 태그

  • <header>: 문서나 섹션의 머리글을 나타냅니다.
  • <nav>: 내비게이션 링크를 포함하는 영역을 나타냅니다.
  • <main>: 문서의 주요 콘텐츠를 나타냅니다.
  • <article>: 독립적인 콘텐츠 블록을 나타냅니다.
  • <section>: 문서의 구획을 나타냅니다.
  • <aside>: 주요 콘텐츠와 간접적으로 관련된 내용을 나타냅니다.
  • <footer>: 문서나 섹션의 바닥글을 나타냅니다.
